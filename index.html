Wijmo Dashboard with Weavy

This app shows how you can use Weavy to add contextual collaboration features such as activity feeds, document sharing, and chats to Wijmo applications.

In this case, the app is a simple dashboard that shows sales data stored in a Google Sheet. Users may sort and filter the data, and the chart is updated automatically.

This app was created to help you get started in the Weavy/Wijmo Hackathon.
